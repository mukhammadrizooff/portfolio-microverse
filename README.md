## 💻 Getting Started <a name="getting-started"></a>

> Just open index.html to open the project

### Prerequisites

In order to run the tests you need node.js installed.


### Install

Install this project with:

```sh
npm i
```

### Usage

Open `index.html`.

### Run tests

To run tests, run the following command:


```sh
npm run stylelint
npm run hint
```
